# Flappy_Bird no garavity
try to remake Flappy bird by python
- [video-demo](https://www.youtube.com/shorts/5MB1kTjaoVA)
## Install: 
```bash
pip install pygame
```
## Play
To start the App, run
```bash
py Flappy_Bird.py
```
# Features
### use keyboard to move:
#### a: move left
#### s: move down
#### d: move right
#### w: move up
